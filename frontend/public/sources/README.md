# Thư mục chứa nhạc nền

Đặt file nhạc nền tại đây với tên: **music-xi-zach.mp3**

Đường dẫn đầy đủ: `frontend/public/sources/music-xi-zach.mp3`

Sau khi đặt file, nhạc sẽ tự động phát khi vào web (nếu trình duyệt cho phép autoplay).

Nếu autoplay bị chặn, người dùng có thể click vào nút âm thanh để bật nhạc.
